# Monallo - AI-Powered Web3 Product Suite

Monallo is an AI-powered decentralized application suite built on Polkadot, featuring innovative products that bring artificial intelligence to blockchain operations.

## Products

### Monallo AI Pay
AI-Powered Intent Recognition & DeFi Assistant
- Natural language interface for Polkadot DeFi operations
- Transfer, swap, stake with simple text commands
- Cross-chain transaction support
- Powered by Qwen3 AI

### Monallo AI Video
AI Video Generation Platform (Coming Soon)
- Powered by Seedance 2.0
- Create AI-generated videos with text prompts
- NFT minting integration
- Social sharing features

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Ethers.js
- MetaMask Integration

## Getting Started

```bash
cd products/ai-pay
cp .env.example .env
# Add your Qwen API Key
npm install
npm run dev
```
